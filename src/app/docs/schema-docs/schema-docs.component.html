Schema

Schema is set of rules that defines functionality of each component/validator/presentation process. We are using the following schemas:

  • JsonSchema - provided by JsonForms, used for creation of form controls, validation etc. We are using JsonSchema7 - you can find interfacace here
  • UiSchema - compatible with JsonForms is used for displaying form controls at the front end. There are several differences with JsonForms UiSchema (more typed options available) which you can find on each renderer docs. Interface is availabe here
  • UiSchema for preview - also compatible with JsonForms but with additional properties for displaying the preview of form value. WIll be available in the future
